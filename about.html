#AboutWeatherWise

WeatherWise is an innovative weather web application developed as a final semester college project, showcasing the culmination of technical skills and creativity. Designed with a user-friendly interface, WeatherWise provides real-time weather, forecasts. The app integrates advanced meteorological data sources to deliver accurate and timely weather information, including temperature, humidity, wind speed, and precipitation levels. Additionally, WeatherWise features asthetic approach to a web page which is both minimal and optimal and a sleek design, ensuring a seamless user experience. This project not only demonstrates the proficiency in web development and data integration but also the ability to address real-world problems through technology. This project mainly focuses on weather data fetched from openweather and dislpayed to the user. the main tech-stack used in this project are:
-HTML5
-CSS
-JAVASCRIPT
-OpenWeather api
